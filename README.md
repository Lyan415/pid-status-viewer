# P&ID 狀態檢視器（手機版）

單一 HTML 檔案的 P&ID 狀態視覺化檢視工具，專為手機／平板瀏覽器優化。匯入由 **[P&ID 狀態色塊工具（離線版）](../pid-status-editor)** 匯出的專案檔（`.json`），即可在畫面上以顏色檢視各 EM State / Phase Step 對應的設備狀態，不需安裝任何 App。

## 這是什麼

這個工具只保留「匯入專案檔＋展示模式」，拿掉了編輯底圖、畫色塊、匯入 Excel 等功能（那些請用電腦版離線工具完成）。適合在產線、廠房用手機或平板現場核對設備狀態時使用。

- 純前端、離線可用，沒有伺服器、沒有帳號、資料不會上傳到任何地方
- 雙指縮放、單指拖曳瀏覽 P&ID 底圖，右側也有 ＋／－ 按鈕與縮放滑桿可直接控制
- 依 EM State 或 Phase Step 勾選，色塊即時變色
- 狀態衝突（多個來源指向不同狀態）會以紅色閃爍框標示
- 含註腳條件（Setpoint／展開表）的 Phase 步驟可手動指定參數值，設定面板固定在抽屜最下方，捲動樹狀清單也不會被蓋住，隨時可以回來改指定的值
- 底部抽屜收合時，仍會顯示一行目前勾選的 EM / Phase 摘要，不用展開也能確認選了什麼
- 可匯出目前畫面的高解析度 PNG，或直接截圖
- 手機縮放/拖曳時會自動暫時降低畫布解析度以維持流暢，放開後自動恢復清晰畫質

## 使用方式

1. 開啟本工具（見下方「如何開啟」）
2. 點右上角 📂 載入專案檔，選擇 `.json` 專案檔
3. 底部往上滑（或點把手列）展開清單，切換 EM / Phase，勾選要檢視的項目
4. 點畫面上的色塊可查看目前套用的來源明細
5. 若勾選的 Phase 步驟含有展開表條件，畫面下方會固定出現「展開表參數設定」，可隨時用下拉選單指定或改選 Setpoint 對應的狀態
6. 右側 ＋／－ 按鈕或滑桿可精確控制縮放程度；左下角「縮放至全圖」可一鍵回到完整視野

專案檔請在電腦版的 [P&ID 狀態色塊工具（離線版）](../pid-status-editor) 中，於「編輯模式」完成底圖、色塊、EM/Phase 資料匯入後，用「匯出專案檔」產生。

## 如何開啟

**方法一：GitHub Pages（建議，手機直接開網址即可）**

1. 到本 repo 的 Settings → Pages，Source 選擇 `main` 分支、根目錄，儲存
2. 稍等一兩分鐘，即可透過以下網址在手機瀏覽器開啟：
   `https://<你的帳號>.github.io/pid-status-viewer/`

**方法二：直接下載開啟**

本工具是單一靜態 HTML 檔，沒有錄音等需要 HTTPS 的功能，下載後用 `file://` 直接在瀏覽器開啟也完全正常，適合離線或內網環境使用。

## 技術說明

- 單一 HTML 檔，內嵌 [Konva.js](https://konvajs.org/)（畫布繪圖），無其他外部依賴
- 不含 PDF.js、SheetJS 等編輯功能所需的函式庫，檔案大小僅約原始離線版的 1/10
- 專案檔格式與電腦版離線工具完全相容，兩者可互通
- 縮放採用 `requestAnimationFrame` 節流，並在互動期間（雙指縮放、拖曳、滑桿）暫時將畫布 pixelRatio 降到 1，放開手約 0.2 秒後自動還原成裝置原生解析度，兼顧手機效能與畫質
